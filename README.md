# HR_Analytics_Dawa

This project demonstrates **Human Resources analytics** for a pharmaceutical company in Kenya.  

**Objectives:**  
- Track **employee demographics, roles, and departments**  
- Analyze **salary distribution and payroll**  
- Evaluate **performance metrics and bonus eligibility**  
- Provide actionable insights for **HR planning and retention**  

**Skills demonstrated:** SQL, HR Analytics, Payroll Analysis, Performance Metrics, Employee Segmentation
